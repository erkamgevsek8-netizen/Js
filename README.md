# 📝 To-Do List Application

Görev takip uygulaması - Yapılacakları yönetin ve takip edin!

## ✨ Özellikler

- ✅ **Görev Ekle** - Yeni görevler ekleyin
- ✔️ **Tamamla İşaretle** - Görevleri tamamlanmış olarak işaretleyin
- 🗑️ **Görev Sil** - İstemediğiniz görevleri silin
- 🔍 **Filtrele** - Tümü / Aktif / Tamamlanan görevleri görün
- 💾 **LocalStorage** - Veriler kaydedilir, açıp kapatsanız bile kalır
- 📱 **Responsive** - Mobil ve desktop uyumlu
- 🎨 **Modern UI** - Güzel ve kullanıcı dostu arayüz

## 🚀 Başlamak

### 1. Repoyu Klonla
```bash
git clone https://github.com/erkamgevsek8-netizen/js.git
cd js
```

### 2. Tarayıcıda Aç
```bash
# Dosyayı açıkça tarayıcıya sürükle veya:
open index.html  # Mac
start index.html # Windows
xdg-open index.html # Linux
```

### 3. Kullan!
- Metin kutusuna görev yazın
- "Ekle" butonuna tıklayın
- Checkbox ile tamamla
- "Sil" ile kaldır

## 💾 LocalStorage Nedir?

**LocalStorage** = Tarayıcının bellekte veri saklaması

```
✓ İnternet olmasa bile çalışır
✓ Veriler silinene kadar kalır
✓ Her açılışta veriler gelir
✓ Güvenli (sadece siz görebilir)
```

## 🎯 Nasıl Çalışır?

1. **Görev Ekle** → `localStorage`'da kaydedilir
2. **Sayfayı Kapat** → Veriler kalır
3. **Sayfayı Aç** → Veriler geri yüklenir
4. **Sil** → `localStorage`'dan silinir

## 🐛 Sorun Giderme

### Görevler kaydedilmiyor?
- Browser'ın LocalStorage'ı engellemiş olabilir
- Private Mode/Incognito'da aç
- Browser cache'i temizle

### Tasarım bozuk görünüyor?
- Sayfayı yenile (F5)
- Browser'ı güncelle
- Cache'i temizle (Ctrl+Shift+Delete)
